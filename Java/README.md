# Sistema de Control de Estudiantes - Java

## Descripción
Programa desarrollado en Java aplicando Programación Orientada a Objetos (POO).

El sistema permite registrar estudiantes, validar notas, calcular promedios y determinar el estado académico de cada estudiante.

## Características
- Uso de clases y objetos
- Encapsulamiento
- Constructor
- Métodos get y set
- Validación de notas
- Cálculo automático del promedio
- Verificación de aprobación
- Mostrar información de estudiantes
