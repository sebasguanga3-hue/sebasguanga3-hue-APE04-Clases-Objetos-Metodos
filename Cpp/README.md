# Sistema de Control de Estudiantes - C++

## Descripción
Programa desarrollado en C++ utilizando Programación Orientada a Objetos (POO).

El sistema permite registrar estudiantes, ingresar notas, calcular promedios y determinar si el estudiante aprueba o reprueba.

## Características
- Uso de clases y objetos
- Métodos get y set
- Constructor
- Validación de notas
- Cálculo automático de promedio
- Determinación de estado
- Listado de estudiantes
- Conteo de aprobados y reprobados
