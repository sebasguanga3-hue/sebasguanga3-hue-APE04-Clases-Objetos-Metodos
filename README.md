# APE04 - Clases, Objetos y Métodos

## Descripción
Proyecto desarrollado en C++ y Java aplicando:

- Clases
- Objetos
- Métodos
- Encapsulamiento
- Validación de datos

El sistema permite registrar estudiantes,
calcular promedios y determinar si aprueban
o reprueban.

## Funcionalidades

- Registro de 5 estudiantes
- Ingreso de notas
- Validación de notas
- Cálculo de promedio
- Determinación de estado
- Conteo de aprobados y reprobados

## Lenguajes utilizados

- C++
- Java
